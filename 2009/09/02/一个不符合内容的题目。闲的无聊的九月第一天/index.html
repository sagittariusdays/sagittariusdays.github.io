一个不符合内容的题目。闲的无聊的九月第一天

发表于 2009-09-02 更新于 2020-08-16 分类于 空の人生

最近晚上真的很闲哪，现在到这个时候还跑出来……
其实是因为蚊子睡不着……就算有一只蚊子就算它不咬我……我还是睡不着。于是，喷了杀虫剂，抱着电脑跑出来。
九月一号，连着看了五六天的死神了，一天宅在家里，除了死神就是做的那把半失败的小木剑，呃，木刀。木头的，话说，还被划伤了一次……
果然这次回来算是什么都没有做。觉得不公正？算了……只要拼，就会有好结果的吧。
下载J文版的战2，有种，但是速度慢。算来算去，已经下载了一百多个小时了吧。好像电脑什么的都没有怎么关过，出去，睡觉都在挂下载。没办法，谁让它体积大速度又慢呢。
又是这个季节，又是一样的，又是这个时候，只有我一个人。
已经习惯了某几个的开学过早。不过虽然只有几天见面，留下的依旧是一笔财富。精神财富。好兄弟还是好兄弟，一点也没变。那个人也依然是我想要超越的目标。不过，以我现在，能赶的上么？
很少有这种感慨了。好像从上高中的时候开始就喜欢写一些恶搞的东西了，所以到后来老师指导作文的时候都是说“缺乏感情”。不过偶尔也会写一些情感迸发的东西。可能，就是因为上高中的时候太孤独吧。
说起来我还有换马甲的恶癖好。虽然换的不是很多。不过有的时候不喜欢以前的ID了就会去换一个。今天刚搞了新的百度，以前的世界以前的记录都过去吧。不变的是时间，还有我的目标。
那么那个目标是什么呢？也许吧……
看来我还是喜欢无双里伊达的结局，独眼龙终于觉醒，讨伐家康。我最喜欢这个剧本。有志向的人的志向是永远不会消失的，也像家康，拼到那种年纪也要取得天下。虽然我不喜欢他。所以用政宗来做头像。也许，志向是永恒的。
不过历史是不能改变的，他还是屈服在家康之下。以前看文章有一句话，“当铃木元信含泪烧掉《伊达幕府条例》的时候……”志向也会因为外界条件而无法达成。但是只要努力，大家都在帮忙不是么？永恒的志向……
还有那句——“独眼竜は、天に昇る！”
这就是结局。

# 杂谈 # 理想 # 空闲
