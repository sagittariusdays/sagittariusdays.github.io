林间道

发表于 2012-11-24 更新于 2020-08-16 分类于 骑行记

接着昨天的步伐，今天去了水之森公园。 其实今天才是事先定下来的计划，昨天从折立回来跟ヒメちゃん商量了一下 七点半从会馆出发。 其实早上出来的时候正在下雪，但还是努力地骑了过去。 额，这种小学生日记的展开是怎样⋯⋯ 总之到达公园的时间比想象早了很多。 其实公园的内容并没有什么好多说的。无非就是树林，水池，水池，树林。 偶尔有禁止进入的地方，但是也毫无感觉地过去了。 拍了挺多照片，但是基本上自己不太满意。 之后这样的时间不多了呢，毕竟我自己基本上只会选择城市的地方，森林对我来说，难度太大了。 其实不知道以后还能不能再见了。毕业以后。 那时候我会在哪儿呢。 我煞风景的能力真是一流啊。

追梦人

发表于 2012-08-25 更新于 2020-08-16 分类于 百味

[youtube=http://www.youtube.com/watch?feature=player\_embedded&v=BtGnryJvpd4\] 去年离开学校之前我把录取通知书和毕业证摆在一起拍了张照。由于当时满脑子已经浆糊了所以构图很傻不放了。 那以后我又离开了。不是第一次离开家乡，也不是最后一次。 这到底算不算去追逐我的梦想，我也不知道。总之信念是有，不知道能不能贯彻下去了吧。 那年以后，我再也买不到90元一张的学生车票。 不仅是因为列车取消，同时，我也没有了学生资格。 那张最后的车票就这样静静地躺在我的钱包里。 另一边，是我新文件袋里的，那张用新的录取通知书换来的，学生飞机票。 这一路走来，谢谢你，一直陪在我身边，陪我追逐我的梦。
