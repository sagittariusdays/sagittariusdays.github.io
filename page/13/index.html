却道天凉好个秋

发表于 2010-10-07 更新于 2020-08-16 分类于 空の人生

算来算去已经有好久没打理过这一亩三分地了。看了看也差不多该长草了。本来之前想过写点儿什么，但都由于这种那种原因或推迟，或遗忘了。直到前天，九儿跟我说，博客好久没更新了吧，于是突然心里暖暖的，感觉是要过来看看了。

其实好久没有过一种感动什么的了，这么些年来这么多个日日夜夜都这么平淡地过去了，也渐渐忘了自己是在过自己的生活，却只是为了存活在这个世界上而活着。刚才突然对我说，你已经失去自我了。然后算是把若干个月中天马行空的我的灵魂突然拉回到肉身之上，很不习惯。不过……是啊，我几乎失去了自我。为了毕业而读书什么的，早已经是生活里似乎是亘古不变的一条真理了，却没有想过，我要为了我自己去读一些什么书。

然后就是像上个学期突然觉得呀图书馆的书好多啊我的时间不够用的这种感觉。人一旦定下一个目标之后很容易被激励起来，同时也容易犯上后悔的毛病。不过老话也讲过，后悔药没的卖，如同火车（OMG不知道为什么突然想起这个玩意儿），开过去总不能随便回头的。

我自道是不怎么文艺的一个人，因为文艺跟我实在没什么太大关联。22岁身高平庸肤色如炭头发蓬松胡子拉茬标准猥琐理工男一枚，但是心里还净想往文艺的边儿上靠。钢琴什么的，设计什么的，标准的空想家。不过硬要靠的话还是能靠上点谱儿的，毕竟，相比这些，我更不擅长的可能还是理工。

总不能因为本科是什么就定下了以后没前途了。毕竟毕业以后不从事本科工作的大有人在。而我也想成为在我四五十岁的时候能和自己爱的人还有自己好兄弟什么的有一间工作室然后拥有相对自由的工作时间的人——可以一段时间不工作，工作起来也可以几天几夜不停。当然还是要我喜欢的工作。想起以前师叔说的，你这个年龄，有梦想就是好的。但我现在想说，不好意思师叔，我还是想走这一条路。

这是我在上海的第四个秋天了。每年的十一九儿总会陪着我，这次也回家了。然后当那天自己一个人空空荡荡地走在本部里的时候，再看着那明媚得几乎要洒出来的日光，想到了我“年轻”的时候，那时候我还有梦想。不同的是，现在的我充实着理想，充实着这几年来的经历，充实着，对即将逝去的大学时光的感伤。

秋日的阳光此时就如同一把刀，一点儿一点儿地把年龄和浮华从我身上刮去。这却正是我最喜欢的秋天。秋天正是平和心境等等的良机。这种赞美辞说出来能有一大篇。然而写的再多也没用。大学这几年，已经让我从躁动激情的夏天，渐渐步入了平和凉爽的秋天了。

心里总会想着一个场景，在以前，它是夏天的林荫道，有着水泥的路面和沙地上的单杠，远处还是碧海蓝天，如今，这些都已不再，取而代之的而是秋日的密林，安安静静的小屋。就是这些。

我也不知道这个算不算一个梦。至少现在我还知道，人，有梦想，并且为了实现梦想去追逐，这种简单的生活就好。

每个人心中的迦南美地

发表于 2010-08-05 更新于 2020-08-16 分类于 仲夏夜之梦

还是把刘耕宏的那个专辑名称拿来用了。在文采上我总是欠缺那么一点儿，逻辑上也是。所以只能这样使用拿来主义。

所谓迦南，位于西亚，也就是巴勒斯坦。这是一块“应许之地”，是流着奶和蜜的土地，是以色列人的圣地。

以上文字摘自百科。

让我对这两个字有如此深的印象，还是刘耕宏的专辑，其中有一首歌叫《甜蜜柏拉图》。歌声可以储存记忆，这其中存下的，便是高三的暑假，还有我刚到上海时那暮色下的天空。至于为什么是傍晚，那是歌的第一句，咖啡色的天空带着微弱的企盼。

我发现我对夕阳有一种特殊的感情，从高中开始。而上海那时的夕阳让我觉得能天天看到这种景色，就跟在梦里一样，多好。然后往事不值得回首。当我从梦中醒来的时候，夕阳已经过去了。没有梦了。但是这种简单幸福的生活，比梦里来的要美好。

每次来本部，傍晚的时候，看到天上挂着的喘息着的太阳，经常会想到三年前那种我认为很“漂亮”的夕阳。然后久不运作的大脑就开始运转，总会把现在的背景替换成橙红色夕阳下的帕提农神庙，由于海风的效果衣服和头发开始轻微飘动，此时背景音乐应该适时地想起。然后总会想，唉呀我们去希腊吧。话总是没说出来，不管是因为希腊的经济大衰退还好，还是因为现在没有能力去做出那样的许诺也好。希腊，总会去的吧。那就是迦南。
