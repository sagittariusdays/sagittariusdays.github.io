期待与沉淀

发表于 2020-06-18 更新于 2020-06-28 分类于 essay

并不浓密的爬满了天空，六月的微风却夹杂着一丝微凉。漫不经心地从电车上下来，走到南大泽站前的Starbucks。

搬家之前的有段时间，每天都会来这里坐坐，甚至搬家之后也偶尔会绕过来，点上一杯当季的饮料，坐在桌前发会儿呆，或写写字，享受那十几分钟的时光。
因为来的次数也多，时间也比较固定，也就和那些时间的店员都混了个脸熟。后来每次点单的时候也会简单地聊上两句，抑或是在杯子上写一些诸如“工作辛苦了”之类的话，这可是在其他的店里没享受过的服务了。店员大多是年轻人，看样子像是旁边大学的学生。偶尔聊上两句，笑一笑，能够感觉到扑面而来的青春气息。纵然知道在这微笑的背后，各自过着自己辛苦的生活。

还记得搬家之后没多久有一次过去，店员招呼我道，有一段时间不见了。确实是如此，而我也只是笑了笑，随口应付了个答案过去了。如果是还住在那里的话，我可能会说一些比较现实的理由，不过那里已经不属于我了，或者说我已经不属于那里了。没有必要多说什么了吧。

随后疫情如潮水一样汹涌而来，从上一次绕路过去享受悠闲的时光，大概过去了有半年了。

自动门一直开着，门口多了一瓶消毒液，桌子少了几张，空出了不小的空间。
此外，依旧。

店员变成了一些陌生的面孔。普通地点单，普通地付款，普通地取餐。目光的交接不用透过口罩，却也是看不透口罩下面的表情。
店里不如往常一样生气勃勃，客人各自坐在位置上，自顾自地做事情。

我找了个空座坐下了，呼吸着熟悉而又陌生的空气，想要溶解在这背景之中，却又如沉淀一样，置身于外面。

今年没能看到那里的樱花了。我这样想着，走出店去。

Prelude

发表于 2020-06-17 更新于 2020-06-28 分类于 essay

今天找关于Neural Network Console的资料，发现了一个Blog。

作者是一位年过六旬的文系SE，看样子是有着丰富的实践经验。

当然我一开始的着眼点确实是在Title上硕大的“文系”两个字上。

读了几个小文章之后发现，博主对于基础知识的讲解浅显易懂，也有很多我需要的，想知道的内容。

也许，正因为是文系，才需要把一些很基础的数学知识作透彻的剖析，才能总结出这样通俗易懂的读物来，也因此让我对文系程序员稍微有了些改观。

虽然大部分内容对我来说并没有什么参考价值而已。

就像我曾经以为上了大学，专业就决定了之后的一切；
就像我曾经以为没有什么好改变的了。

世界是不停的，人也不会停下，那么总要有些东西，陪伴着我前行。

是时候继续学习了。
————我第无数次发出这样的感慨。

随笔：百日谈

发表于 2020-06-14 更新于 2020-06-28 分类于 essay

疫情随想

在疫情的影响下已经过了半年，在家里工作也有三个来月了。
这半年过得是那么漫长，以至于在回忆疫情之前的时光的时候，仿佛是蒙上了一层灰色的滤镜，遥远又虚幻。
窝在家里与社会隔绝的这百余天里总会胡思乱想，然而瞬间转逝的想法难以捕捉，心里燥燥地也无法记录下来。
如果能够沉淀下来将所有的想法全记录下，怕是也能写一部十日谈之类的作品了吧。
然而过去了的也拉不回来了。

所以算是再开了一个Blog，来记录这样的想法。漏网之鱼肯定有，不过能记多少下来，对于今后的自己也是值得的。

希望做到的事

手账也已经开了几天的窗了。大概是从亚非文学bot退博之后吧，摘抄的内容一下子少了不少。
不过上面也有不少自己的想法，留着记录下来也未尝不可。

隔一段时间写一些文字，或多或少，来练练自己表达；
专业上的东西做上足够的整理，还有专业关连的需要学习的知识，不总结的话总会忘掉；
再就是兴趣爱好，剪报一样的看看做金木石的东西；
还有自己网站的美化；
大致就这几类吧。

希望一年后，五年后回头看的时候，能够切实地感受到自己的成长。
