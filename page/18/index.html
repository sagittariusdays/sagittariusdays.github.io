写在食过天不晴之后的一大堆【闲言碎语】

发表于 2009-07-22 更新于 2020-08-16 分类于 空の人生

一回到脑前面对的是多少诅咒这鬼天气的话。话说一连数日艳阳高照一直到昨天下午还是晴热高温没有云。然后昨天晚上就突然下起雨来了。而且这雨还一直下下到了今天这数百年一遇的日食下面。还好到现在它还没晴，不然的话，嘿嘿……不过就这样，三百年一遇的日食就被这三百小时一遇的雨给OVER掉了。
想想数年前还是那个小孩子的我躲在树下看月食的场景。月食比这要长，长，长……然而那天，之前也是晴空万里，一到月食的时候却布满了云。不爽呀不爽呀不爽。
这两天的感觉很好，像是真的到了某个夏天。虽然很累很累，但却有了那种期待着的感觉。看了一些关于以前的东西。那时候还是很幼稚。不会再回来了吧，我对夏天的期待，还有那三百年一遇的日食。

忙里抢闲

发表于 2009-06-13 更新于 2020-08-16 分类于 空の人生

考试周快到了，6月1号把6月份的日程表列出来，一看吓了一跳，考试排排站，对于传说中“将成绩开根号再乘以十还有40%挂科率”的科目，我觉得没什么好说的了，那科的考试居然夹在中间，前一天一科，后一天一科。算了算，不每天数小时地花在那上面不行了。然后再定睛一看又傻了：排在前面的居然还有三门考试，居然还都不容易，居然我只有两个星期复习这五门会搞死人的科目。哈，凤凰涅槃，浴火重生。本来想把这篇杂文叫“忙里偷闲”的，后来想我这种在早上来图书馆抢了座然后开始写博这种行为简直称不上“偷”，而是“抢”了。而且话说这学期的选课，必修课都选上了，还可能跟08的小朋友一起去上。到6月29号考完最后一门，7月6号开始实习。还不知道实习去哪呢，据说是一个空调厂，夏天去那里……至少算是一种享受吧，然后一直要实习到15号，18号搬校区，有种流放边远的感觉。然后纠结的是，22号有一场很大很大的日全食，上海正好是最佳观测点……那这么说的话，我要什么时候回去捏？复习啦~

Summary@季·2_写给夏天

发表于 2009-05-09 更新于 2020-08-16 分类于 再不再四年

立夏之后，我才有了春天的感觉。

一如既往，当我真正的想写点什么的时候，打开WINWORD，却不知道该说点什么。于是，本来在路上计划好的文字都变成了零乱的片段。既然又是这样，那，还是按时间顺序来吧。

其实从3月底考完二级的时候就想要写点什么。但是，那个周末是一个凌乱的周末：而且这之后的几个周末一直是凌乱的周末。在进入考场前几分钟，我发现手机没了，那天是28号。不安地考完了一天，然后得知消息手机在别人手里，要花300块赎回来。倒是，二手的6300你买也花不了几百块钱，当时我很有一种冲动对着那人说：你丫撕票去吧，爷不管你！不过还是出于经济原因，跟那人取得联系，然后把交赎金的时间地点都约好了，那厮消失了。于是，只好不管经济原因，迅速向家里报告，次日中午，我就拿到了现在的这部手机。以上是3月末丢手机事件的始末，直到现在才写出来。

四月。又是一个不知道怎么了的四月。天气中凉，还有心情。好像是从网上下下来了蒋以前一直念叨的战2，就把时间双手奉上。然后，在月末的时候，显卡终于不负我的期望崩溃了。同样也是28号。依此推断，5月28号也会发生点什么。我做好万全的准备。中午就烧掉了，等我想起来还有客服这么一个东西的时候，早过了下班时间不知道几小时了。于是：29号早上翘课，客服八点半上班，我八点三十一的时候一个电话过去，她告诉我昨天（28号）刚刚过保修期。然后我又冲动了。还好客服姐姐跟我说因为是昨天（保修期内）出的故障，所以会派工程师来。不然现在我写这些东西也就失去了物质条件。30号早上十点工程师的电话来，我毫不犹豫在E文课上请了假回来。一个小时后，我的“物质条件”变成了一堆电路板和一堆塑料板。工程师说是保修，换了主板和显卡没要钱。刚刚从家里申请来的修理费用就这样我塞进了钱包。

第三件大事没有按我的预期发生在5月28号。4号刚开学我要去把多下来的钱存到银行里。悲剧又发生了：到了银行站到了自动存款机前打开书包翻了个底朝天却怎么也不见钱包的影子。在我们把今天我所有去过的地方及所有去过的地方的失物招领处都访问一遍之后，我得到结论：它丢了。可怜我那数百大洋，和银行卡，和身份证。这三样东西一起丢……真是傻到可以。当晚挂失银行卡，之后一天无尽地徘徊在若干派出所之间。我很不明白为什么办个身份证要我在派出所A开出证明，在派出所B打印所谓条形码，然后到派出所C去照照片，再回到派出所B去交钱。一天折腾死我了。幸运没什么大损失。

以上是最近发生的三件大事，仅此三件，就把我搞得焦头烂额。还不要说其他的事情了。

其实应该从这起一篇新日志的。但是，也没什么要说的了。

在春夏之交想起春天，那什么时候会想起夏天呢？

今天又快速看了一遍秒速五厘米。依然好喜欢那种画风，那种看似轻松却又让人喘不下一口气的故事。看完心却隐隐作痛。好吧其实我不是一个多愁善感的人。看到最后一部结束，听着那看似扭曲的歌声……我在低头。小马说我在哭。事实没有。看此类的东西，我是不会哭的。表面上。心里是另一个我。故事是有点小消极，也很感人。但我的触动却不是因为明里和贵树一次又一次的错过，不是贵树在现实生活中那种无奈，而是我真的知道，这种感觉，还有我一直所梦想的夏天，都已经彻底地离我远去，消失不会再见了。

似乎不应该发出“我的童年就这样结束了”这样的感慨。但20岁的确对我来是，是一个尴尬的年龄。我依旧留有童年的那个梦，那个让我朝思暮想了快十年的夏天，无法用语言表达，甚至我自己也不知道是怎么样的一个夏天。它曾经是多么近地来到我的面前，但它也永远无法再来。一年年的现实让我一年年的放弃这个梦。家里的天空一年比一年小，一年也不如一年清澈；而我的天空，现在只要有阳光，就足够好。

许多年我得到了许多也放弃了许多。至现在唯一不变的依然是GC。那天突然发现百度上居然有GARNET吧，瞬间有了一种归属感。真的是归宿的感觉。不会再有哪些歌让我刻下如此多的记忆，更不会有哪些歌让我再次回到那个还在做梦的年代。YURI的声音，NANA的词，都已经成了那个年代无法抹掉的记忆，还有，录入梦想。一如我在一遍又一遍地听HOLY GROUND的时候，总会浮现出忘れ咲き 中，那些回忆的映像。

初/杂语

发表于 2009-03-04 更新于 2020-08-16 分类于 再不再四年

算是学期开始吧。虽然已经有一个星期了。
来了上海以后一直没时间好好整理自己的东西，走之前怎么乱的还是怎么乱的，归置了一下就重新用起来了。大概算来，好像已经有十多天了，一直是阴雨天。下雨总不会是太好的事，尤其是在上海，尤其是在二月。这十多天冷的要命，一直在后悔没有把家里的保暖衣服带过来。人是算不过天的，在我以为是一个春暖花开的地方却是这样的寒冷。失策，失策。
这个学期算是好多的开始吧。现在每每想到上个学期总会让我不爽。反正过去的就是过去了，上海也有我所等待的事情，假期的北京生活也给我造成了不可磨灭的影响。不管是什么，我觉得到了要改变的时候了。不过，以前也这么想过，结果呢？我不知道。我也不知道这几年的大学生活会给我带来什么。无论怎么说，都要有一点动作了。毕竟我害怕的，还是寒假里说的，到我们再次见面的时候，没有什么好交流的了。
