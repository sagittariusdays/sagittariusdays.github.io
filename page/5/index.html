随记

发表于 2019-09-03 更新于 2020-08-16 分类于 uncategorized

列车从景点旁边呼啸而过，穿过隧道，玻璃上映出我沉默的样子。

我平时是很少有一个人出行的机会的，不知道为什么最近却很多。然而就算出行，路过一个又一个的景点，也却开心不起来。

这样笑不出来已经很久了。虽然并没有什么大的变故，无灾无病，收稳定，却找不出年轻的时候偶有的那种兴奋感了。回想一下大概是从我越来越“懒”开始吧，干了完了一天的活，回到家里只想躺在床上什么也不干，一个人过一个周末的话就是典型的废人休息时日：睡到很晚才醒，随便吃点东西打开游戏，一个美好的下午和晚上过去，熬夜，累了就睡，如此往复。

然而自觉这样是不行的，总是会想着去做点什么东西改变这样的生活。学习计划列了不少，每次都是列出表来，准备行动吧，然后就再也记不起来，直到下一次列表。记事亦是如此：知道需要写一写才可以练好自己的表达，但是也是迟迟下不了手。

偶尔在各种SNS上面见到朋友们晒出各自的生活，我就总会不由自主地和周围的同龄人去比，A年入百万，青年才俊，B家庭和睦，生活美满，每个人都有值得我去羡慕的地方，在我内心的某个地方我会把它们收集起来，转化成越来越重的石头，压在胸口，越来越感到不安。

更何况难处理的人际关系。我自身不擅表达，遇到冲突的时候亦会更加沉默，如此这般，反而加深了个中的裂痕。

随之而来的危机感是不会等人的。

成功的人会把这种不安转化成自己的动力前行，我也该如此了。比起累和辛苦，我想我还是更怕这种被人远远甩在后面的感觉。像沉入深海，无法浮上来。

偶尔也会跳出一个声音来跟我说我这个样子已经很好，不需要去那么努力。我也会好好想想，看看我平时没有看到的那个世界，的确是这样没错。然而当我的目光又回到我之前关注的优秀的人周围的时候，那种如山的压迫感又会袭来，它们是没有感情的。

似乎是这种时候，我只能选择逃避，或者战斗。

前者是我一直以来的对策，然而这样下去，压在心头的黑云永远不会消去，每次看的时候就会发现已然越积越多。

那么，就只有战斗了。

2019/9/3 于前往金泽途中。

旅行的意义

发表于 2018-03-20 更新于 2020-08-16 分类于 uncategorized

已经不记得上一次有计划的旅行是多久之前了。 我自觉是一个比较死板的人，死板到旅行的时候也会做好计划，如果到时候计划没有顺利执行，多半会烦躁起来。 但是总归心里还是的些小调皮的，经常也会想去踏上一次说走就走的旅行，然而现实告诉我，你既没有时间也没有钱，还是省省吧。 那我就省省吧。 省归省，但是内心里还是会泛起涟漪的。打发掉这种感情的最好方法无非就是地图旅行了。打开Google，四处把鼠标拖来拖去，就好像自己真的去环游世界了一样。 每当谈到哪里有什么东西好玩，我总能说上一堆出来，等到别人开始惊讶问将起来，我却只能挠挠头，不好意思我没去过。 那你说个球啊？ 大概是因为这样会让我有满足感吧。 我没去过希腊却向往那边的白房子和地中海；没去过君堡却仿佛看见过金角湾数百年来的风雨；就连巴黎柏林维也那这些名地，也只能从书上看看她们的过往与现在。 呵，怎么都那么远。那近的呢？ 倒是前几年有过几次去韩国的机会，反而给推掉了。因为去那种语言不通的地方，总会有些担惊受怕的。 然而又有什么好怕的呢？ 我不知道。没有吧。 多半又只是为自己找一个让自己心安理得的借口罢了。 如此想来，以往的旅行也多半都是在自己的安全区内。提着个相机跑来跑去，看看平淡如水却总能让各地而来的游客说成是美景的地方按下几次快门，如同吃阳春面一样没意思。 然而就算是这种程度的出游，在日后某个日时刻翻出当时拍下的照片，一点一点看过去补完当时的细节，却是别有一番风味。那阳春面，已在不知不觉间酝酿成一碗浓郁的豚骨拉面，散发着诱人的浓香。 最近常在朋友圈里看到这位去了哪里那位又去了哪里，发出来的照片无不透露着满足的微笑。我去的话多半是留不下自己的照片的吧。把自己身边所感的环境记录下来，回味的时候，同样又能把自己扔回那个环境里。每个地方都有它自己的故事，如果不了解故事的话却了则没有意义。感受到周围的环境事物千百年来历史的积淀和自然的流淌，这对我来说，应该就是所期待的旅行的意义了吧。 不过翻着朋友圈，内心还是会羡慕的。 但是在准备万全之前，我还是就这样继续我的地图旅行吧。 2013@Fukaura Sta., Aomori, JP 2018/3

似水的流年，过眼的云烟。

发表于 2013-07-29 更新于 2020-08-16 分类于 再不再四年

本来是准备写成封二的，但是不知不觉跑偏了，就放到日志里吧。 封二的话一般是用来写作者介绍的吧。但是这又不是本书，况且即使是本书，我也不是一个合格的作者。不过不写点儿对不起这片空白，虽然这里的标题就是一片空白。 那么就来说说头像吧。 照片摄于2011年5月。 简洁如此的介绍。为我拍照片的那们朋友，我想，对于彼此来说，我们已经互相成为对方人生的一位过客。也许在之后的人生中我们永远不会再见面，但是每当回想起大学毕业那个樱花飘落的季节，看到这张照片便会想起，曾经有那么一位朋友，我们一起玩桌游，一起打球，一起看电视，一起打游戏，一起拍照。这里是拍照，是因为当时我的技术根本不能称为摄影吧。虽然现在也不能。 就这样我怀着对之后新生活的憧憬，愉快地度过了那个春天。虽然我知道，这个朋友，以后可能永远不会再相见。那象征大学生活完结的假期来临，我离开了上海，走时互相留下了QQ。只是那个头像，没有再闪动过。 说起来其实当时我们根本也没认识多久。他是我当时的女友的朋友的男友。虽说是从大四下学期才认识的，但是之前也见过好几面，也都互相听说过对方这个人，只是几乎没有交流过而已。见面也都是在女生寝室楼下吧。他在学校附近租了房子，之后也请我们过去吃过饭。后来去的次数多了，开始熟起来。有段时间几乎周末都会去他那里。还有其他的一些朋友。那段时间很流行桌游，于是我们就买了好多种，三国杀，风声，等等还有我记不得名字的许多，现在也都留在那里，不知道看到这些，会不会怀念起那段欢乐的日子。 开春的时候学校的樱花要开了。得知我想去拍照，他便来教我摄影，于是有那么几天，我们经常往旁边的公园跑。当时公园里经常看见一个爱好摄影的老人，我们去了几次，也便混得个眼熟。知道我在学摄影，还特地来跟我聊了聊天，教了教我一些非常简单的东西。总之在那一段时间，附近的街巷也跑了许多，总算是在毕业前留下了一些，算是我当时在那里生活过的痕迹。在樱花开了的某一天我们去学校里赏花，我正在拍学校里懒洋洋的猫，他便给我拍下了这一张。应该是我唯数不多的照片中比较中意的一张，正如我现在，一心想去记录下来某些东西，但终究却留不下什么。当然最重要的让我喜欢这张照片的一点，应该是没有露脸吧。 春夏之交的时候我们一起在那里打过NBA 2K，大家都喜欢篮球，但是我仅仅是停留在喜欢的层面上，每每在电脑上打都会被虐的体无完肤。后来，我跟着他去过他公司里每周的篮球活动。当然我只是过去混数的，打的不行。也就是那一两个月，我知道了体育场里打球是什么感觉，也切实感受到这刚刚到来的生活，会随着毕业离我远去。一如我后来，跟大学同学在华师大体育场打过的最后一场球，带着笑容，忍着泪水。 进入暑假我便离开了上海。由于出国要给教授带一些小礼品，便托他帮忙捎一些茶叶。他有朋友是做茶叶生意的，之前也经常在他那里喝茶。我和女友在九月份的时候又回了一趟上海，顺便去他那里拿了那些茶叶。确实来说，那应该是我们相见的最后一面了吧。临别的时候说，反正QQ是留着的，以后有事了随时联系，要茶叶了也可以跟他说。但是如之前所说，那个头像再也没有闪动过。甚至在刚才我进分组里想去看看近况，找了半天却不知道到底哪个才是他。如果我不写这篇文章的话，估计这些事情也会淡淡地走出我的记忆吧。找到他的资料，翻了翻，看到他的地址依旧写着上海，工作状态变成固定工作，心里便知道，他应该还在那个城市，过得应该还是很好。 毕竟我们只是互相人生中的一介过客，用不着感慨什么物是人非。认识相处也不过才数个月不到一年而已，在我的朋友里面，也确实不是给我印象最深影响最大的一位。对于朋友之间的关系，平行线和相交线的理论我觉得最能说明吧。像这样的两条人生轨迹互相交叉，又互相远离。终究有些事情是回不去的。 在那之后，他和他女友若即若离了半年一年，最后好像还是迫于家族的压力断了联系的样子。我和我女友两个人一起出国，和国内的朋友联系也日渐稀少。曾经留下过的记忆会淡化，没有什么事情提起来的话便会在脑海里躲藏一生吧。也许当某年某月和朋友坐下闲谈至此，才会打开关于这件事，这个人的记忆的闸门。只是这样的机会，应该不会再有了。
