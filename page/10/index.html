清晨的鸟鸣是我度过黑夜最美的歌

发表于 2011-05-29 更新于 2020-08-16 分类于 再不再四年

为了毕设已经通宵一个多星期了。我知道我自己为什么这么苦逼，我想，我的生活的这一页，最终还是被翻去了。迎面而来的是真正的一片空白，画还是写，我不知道。我用四年的时间写下了现在的这页纸，满满的，没有留下一丝空白。书毕翻页时蓦然发现，满满一页纸上竟没有什么内容。它本来是白纸，现在却变成了黑纸。似乎随着这一页渐渐的变黑，肚子里的墨水越来越少。细细想来，这四年的时间竟没有好好读过几本书。我的爱好呢？我的理想呢？你们都哪儿去了？我找不见。

最近一段时间我喜欢上了熬夜。深夜两三点钟的时候正是万籁俱寂的时候，不用管其他人，这个世界上只有自己呼吸的声音和敲击键盘的声音。偶尔还会伴随着不知道哪里传来的轻轻的呼噜。那是一种“我就是这个世界”的感觉，像毒药，每次经历都会更上瘾。在这深夜里如果没有事情干，沉思便是最好的选择。总是在一个人的世界里才能静下来去想某些东西，这可不比白天。白天不知道哪里传过来的噪音总会在脑子里转啊转，颇有余音绕梁的架势，让人不得安生。然而很不幸，我最近熬夜都是有事情做。全靠拖延症的福，忙到吐血的毕设，这让我知道了我还是可以高效率地去做事情的。人都是逼出来的，像我这种之前不到deadline不会去忙事的人，偶尔吐吐血是最好的惩罚。

总是这样的事情经历下来，总会觉得自己在夜深人静的时候有着超高的效率，从而日渐懒堕起来，总想着“哎呀白天反正是做不下去晚上还好有效率一杯咖啡盯一天”这样的混蛋话，而念叨的结果就是眼里血丝满布，看什么东西都像罩一层红布。

再回到刚才的话题，人总是会在黑暗的时候才能去反思自己。这也是为什么很多沉思室冥想室一点光线都没有的原因。我想，如果有光，便能区分自己与周围的不同，而只有伸手不见五指的时候才会真正融入周围的环境里，那个时候，我才是我。黑暗中的自己才是真正的自我。性恶论便是这样，人之初性本恶，任谁都是为了洗清自己的罪孽而活。黑夜才是人恶劣伪善一面爆发的源泉。

这四年来我一次都没有反省过自己。于是四年过得就是一种没有约束的浪子生活。该吃吃，该玩玩，该睡睡，该不学还是不学。人们都说大学四年并不是来学习知识的，而是来学习做人的。而这样生活下的我，不仅做人不像样，知识也一点没上心。之前我的爱好呢？没了。那之前我所了解的东西呢？忘了。那我的这四年的光阴呢？完了。自甘堕落所导致的必然结果就是道德品质的败坏。最近发生的事让我觉得我离那个境界已差不远了。现在还正在挣扎的边缘，虽然还是不是很确定以后如何，还是要感谢那位点醒我的人。不然我可能以这个样子一直脱线下去。

都说离开才会感到珍惜。经验上来看的确如此。昨天在最最绝望的时候大哭了一场，就是那种突然感到天塌下来了所有人都离我而去了那种的感觉。不过能有这种感觉，还是自己心理太脆弱的缘故。我总觉得眼泪并不是能洗净心灵的，但这次我错了。以前也哭过，并且次数不在少数，但是像这样撕心裂肺地去哭，应该是好多年没有的事情了吧。完了坐在那里，总觉得心里不再是那么回事了，一种脱胎换骨的感觉油然而生。虽然我不知道这会不会是真的，但是我希望就这样，从此告别大学的那个我，翻开新的一页。

我写这篇开篇的时候刚好天蒙蒙亮，刚开始有鸟叫。昨天痛彻心扉地大哭了一场，现在看什么都觉得是干净的。当然，自己内心的污秽尚未除净，这个还需要一段时间吧。听这清晨的鸟鸣也能听到心里去，看着阳光平行着我的眼照过去，突然觉得我呆在这个经常不见天日的小屋子里，或许对我今后的生活是一种折磨。如果出去了，说什么也要找一间清晨能照到太阳的屋子吧。虽然这样喜欢黑夜，我还是离不开阳光。

然后，现在的我，还是一页空白。

过去，再见。

我的大学，再见。

梦中的大山，压得现实喘不过气来

发表于 2011-02-13 更新于 2020-08-16 分类于 百味

昨天送走了爸妈，看着他们的背景，觉得时间真是摧毁人的利器，不知不觉父母已经到了这把年纪，感觉心中酸酸的。现在最大的愿望，就是能希望他们平平安安，健健康康，好好享受一下以后的生活。

晚上回来做了梦，自己在一片废墟里，不停地走，不停地找人。心中油然而生的一种荒凉。应该是看到那种悲催的景象所以感到荒凉吧。梦中有一种很无力的感觉，想找谁都找不到，平时最亲的人也不在身边。但是看到废墟的时候想到以后这里会重建，变成一片全新的地方，隐隐地又有一些小兴奋，我是不是很变态……最后不知道为什么还要因为这个事情去写日记。有蓝色的笔和金色的笔，我实在是搞不懂它们出现在这里的意义。但是当日记写完时，废墟还是废墟，但是里面突然多出了很多生气。这也算是一种 good end 吧。

我讨厌那种荒凉的无力感。但是又有点儿希望在它的上面创立新世界的期待。我知道我的想法，但我不知道我能不能做到。对于未来来说，这也是一种无力感（什么逻辑……），真让人讨厌。

看那不是飞走的少年时光么

发表于 2011-01-26 更新于 2020-08-16 分类于 再不再四年

打包了所有东西，寝室里除了我一个人也没有。桌子上面盖着各种报纸，床板上也是空空荡荡。

这不是第一次回家了，不知道为什么拉出箱子锁门走人的那一刻，觉得有一种“马上就要从这个地方滚蛋了再也不属于这里再也不会回来了”的感觉。真是讨厌啊。

在空荡的学校里走着，突然感觉这又不像冬天了。一路上总想着大一的时候听到的那句话，一月二月的时候就不冷了。这句话给我在上海的每个春天都留有了幻想。这不像是冬天的阳光照在身上，说温暖吧倒也还行，多少给了一点春天的感觉。但同时背景里的不和谐因素在响着：喂你怎么这个时候还赖在没人的学校还不回家。之类的话。总之，这种感觉不像我在上海见过的。不像我在任何一个大城市里见过的。只有在家里那种小城，那种春天还没有来的时候微微躁动的气息，才跟这个样子最符合。但是，我却要滚蛋了。

明年的这个时候我不会再在这里，具体在哪里我也不太清楚了，看自己了吧；也不清楚跟现在身边的人到底会隔多远，这个要看所有人的努力了。不过，更好的总是会来的，不是么。就像我每天在那个小城里骑车飞奔喝两毛一瓶的汽水的时候，从来没有想过自己现在会站在这样一个舞台上，虽然不是以什么精英的身份。

顺便说一句，怀旧总是人之常情，年轻的感觉真好。
