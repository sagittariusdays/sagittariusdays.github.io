Summary季·1

发表于 2009-02-18 更新于 2020-08-16 分类于 再不再四年

从北京回来，今天已经是第二天了。

这一天的生活依旧，就好像我从来没有去过北京一样，早起，睁开眼打开电脑，然后就是对着屏幕不定时间的发呆；吃中饭；下午依旧发呆；再吃晚饭，晚上依旧发呆。这样一天随着吃饭时间随机地划分成三个几乎等长的时间段，相同的是这三个时间段我都在干着一样的事。

想起字在高中的时候对我说的一句话，人之初，性本贱。我可以用这句话来解释这样一个事实，就是还在放假之前我就想着要回家，然后在对诸如“回家可以大吃大喝”“回家可以联帝国”“回家我一定好好看书”此类的幻想中度过每个学期的最后一段日子。刚下火车的时候是很激动很兴奋，而且上面三个目标我已经完成了三分之二。不用说也会知道哪个没有完成。然后每到假期的最后几天，就像是今天这样的日子，我开始特别想念学校。倒不是因为别的什么。有的人是因为考试没过要早去补考，有的人是为了“家”业早去陪G/BF。我这两种都不是，只是单纯地想回去，到了那边可以脱掉重重的棉衣换上春天的衣服，或者去图书馆：即使不学习，站在9楼看看窗外的风景也是很不错的感受。既然回去的理由这么单纯，那么我回去这件事情就可以完完全全地用“人之初，性本贱”这句话来解释。但是不幸的是，我早已经过了“人之初”的那个真正单纯的阶段。我已经是一个奔三的人了。

这个假期还有一个变化就是我从二叔口中所说的那个仿冒的“网络视频男”变成了一个真正的“网络视频男”。本来没这个变化的，在北京那几天最初只是上上网，打打帝国2K9就好了的。本来这个活动是可以贯穿整个假期的，但是我还是在北航受到了松哥的感染。忘了从某一天开始，突然想到了原来有一个叫优酷的网站，然后就搜了几个以前听说过的动漫来看。最初是To loveる，看完以后我已经挪到了北交。在再次确定那里能联上网以后，就开始看“凉宫春日的忧郁”。凉宫看的很有感觉，说老实话我是比较喜欢这个风格的。它比较接近我的夏天。虽然距离依然那么遥远。然后管晏姐姐要了一点所谓“治愈系”的动漫，还没有来的及看。胡瑞涛说，老看这个不好。原话我忘了，意思好像是会被漫画腐蚀。然后他列举他自己的经历来告诫我。我想想也是。开学以后不再这样天天看了。在北京那些日子给我的冲击是巨大的。似乎我的世界观被彻底地洗了一遍。具体怎么样的变化，说也说不出来。但最直接的还是，以我现在的努力状态，以我现在的水平是远远达不到要求的。我也不知道要求到底是什么。但是，至少在同时跟我一起去学的那些认识的人中，胡瑞涛，松哥，小李，当他们出国或者出国回来以后，我不想蜕化到跟他们一点可沟通的地方也没有。不想这样平凡地注定这一生。在那的二十天之于我，就像那一场棒球赛之于凉宫春日，不同的是，她能够改变一切，而我不能。这也是看漫画的悲哀。似乎幼年的一切正在离我远去。

自从我踏进大学校门的那一刻，我的生活改变了。有好的方面也有不好的方面。但就这一年半的成果而言，似乎是不好的占据了上风。再加上年前的连遭背运，让我的心情一时间降到了谷底。不过，再差的心情也会好转。北京的二十天像是净化水嘴（虽然比喻相当不恰当），再根据人品守恒定律，09年一定会好起来的吧。虽然依然不会再去看新的柯南，至少在我踏入大学校门的那一天起就没看过新的，因为我不想看到结局，不想破灭年少的我在柯南的世界中的那个虚幻的梦；虽然可能再也找不到童年最美好的回忆，那是一个夏天，也是很多夏天，无法用语言表达，但是会一直留存在我的心里；虽然可能永远玩不到那些让我最最接近我的夏天的那些游戏，心跳回忆，青涩宝贝，还有跟这两个毫无关系的SIMCITY……明天，依然会是好的。至少现在，我相信。

再回想上大学前看过的《草样年华》I和II里面的内容，我希望我不会是书里角色的同类。我负责对我的生活起到积极的改变。又来了一年的春天，这个春天似乎比上一个春天要好上不少。而我，则希望这个春天，能给我带来最接近我梦想的夏天吧。好像在上大学前的那个临别文章里写到过，我不擅长在电脑前面码字，有些话面对电脑的时候就是想不起来。我还在想，那时候的我，也是刚刚脱离青涩的一个少年，而一年半的大学生活让我可以在三十分钟内写下以上文字。可能是我今天真的有话要说，也可能是很多很多的话都积攒到了今天。还记得前几天装扮这个BLOG的时候我问胡瑞涛，你平时都怎么才能写那么多？他好像没有给我正面的回答。然而今天，似乎明白了。充实的时候感到的极度空虚，或者是空虚时候产生的极度充实，都会促成我写下以上这样对我这个不擅文笔的人来说的大笔的文字。这样看来，对于尚未进入大学的我来说，是一个积极的改变吧。

希望明早起来看到的是睛朗的一天，希望三个月后，是一个更加接近梦想的夏天。

那些幸福的小日子

发表于 2009-01-19 更新于 2020-08-16 分类于 百味

这次放假回来，感觉就明显不一样了。就算世道再怎么样，我也不能管他们爱干嘛干嘛去。电总归还是要充的，玩自然要玩。不过，学期中间玩了这么多了，现在玩什么也觉得没意思了。本来以为回来了，就可以天天想干什么干什么，想去哪去哪。结果回来以后发现不是那么一回事。早上起来，也不迷糊，每次也都八九点了。大冬天的哪都不想去，就坐在桌子前面望着太阳从东南边一直滑到西南边，数日子一天天过去。快过年吧。
原来每个假期都会天天出来玩，现在也不知道什么可玩了。刚才在回家的路上就在想，到底什么事情是有意思的呢？逛街？还是打游戏？我现在不知道。新的一年又来了，希望新的一年有一个好的开始吧。

半年记

发表于 2008-11-05 更新于 2020-08-16 分类于 百味

不想总结这半年以来到底发生了什么，明明细细林林总总写下来，我也没那个耐心。毕竟来这里看的人可以说是几乎没有，只是写给我自己，权当日记。
首先还是要说的是，今天是我20岁的生日。果然，我把主要联系方式上的生日日期都去掉了，这个世界就清静了很多呢。我本是不太愿意过生日的。回想十八岁生日那天，我自以为过得很低调，不过还是到了晚了，收到了打字，廉头等人的纸巾形式贺卡后，心中也不禁一丝颤动。现在想想，是感动多一些。关于这个，不想多说。一年一年过去，每个数字都有不同含义。20，意味着我的责任，我的成长，我已经在那一段人生最多姿多彩的岁月里，越走越远……祝我生日快乐。
言归正传。大一一年里，除了疯玩就是疯玩，到了这回开学才发现成绩惨不忍睹。到了假期立志要做一个好孩子，结果不到一半就又堕落了。虽说学习是比上学期用功了，但也不能说是努力。学习这里就带过，反正也没什么好说的。
刚来到上海两个月，一个月回去了一次，这是怎么也没有想到的。而且天气越来越冷，两地温差越来越大，这次回来果不其然，感冒了。宿舍里也很乱……还是我懒得收拾。最近的生活有点不规律，可能是玩所占的时间太多了吧。看书的时间也没有了，还不敢去图书馆。可能是对那个地方有阴影吧。
这次回家看了部有关军人的电影，突然发现自己还有那么多事情要做。我不是一个闲人，但一直以来我把自己当成一个闲人。不能再闲了，大家也都在忙吧，小李也是，天酬已经好久不见了。
有的时候觉得自己挺没写东西的才华的，写不了多长还总是虎头蛇尾。肚子里的墨水没了，今天就先到这里吧……
