明天的太阳是最美的

发表于 2012-12-07 更新于 2020-08-16 分类于 空の人生

今天开了一天的zemi，最后的时候发生了大地震。 来这边以后经历过不少地震了，这次应该会留下最深的印象吧。摇了一分钟，坐船一样。 会议室里警报声此起彼伏，大家都坐在位子上，还有的躲到了桌子下。 预告的震度三，事实上有五弱，还是意想之外的长时间。 之后第一件事就是刷微博，说这边地震啦，兴高采烈地。 然后和大家聊了起来，很多人来问情况，很温暖。 那是我来这边之前，311前两天有一次一样大的，里氏7.3的地震。于是有很多人担心，之后两天会不会也出现大地震。 正好最近世界末日留言很火热，也有看到预言说12月7号会大地震。虽然规模预测错了吧【笑】。 总之不是什么大不了的事情了。我自己也不是很信那个来的。 但是之后的事情谁能预测呢，就像去年3月9号一样。 从研究室出来跑去Don Quixote买了点能保存的饼干，万一有什么状况了还能顶几天。家里刚买了两箱水，应该也够用。之后就是买救生包了。其实身边很多能用的东西，随手能拿到。 我还是希望平安无事的。 晚上母上找我聊天，没提地震的事，想了想应该是在学校忙了一天，完全不知道有这么个地震。 于是我也没告诉她，省了担心，毕竟现在没什么事情。 当然希望归希望，真的有事的时候，可能求救都叫不出来吧。 如果之后真的有了大灾害，停水停电，信号不通，联系不上，自己在这边还好，亲友肯定会担心死了。 那么就这么说一下吧，万一遇到什么事情，灾害什么的联系不上，我都会好好的，不用为我担心。我会尽力和亲戚朋友们取得联系，也会努力去让自己让妻子没事的。 如果那天联系切断了，希望有人能看到这里，并且帮我告诉大家我很好，不用担心。 晚上又废话那么多，也不过是个杞人忧天的事情。没有什么大不了的，也没有什么比明天的太阳更美丽的东西。 晚安，世界。

林间道

发表于 2012-11-24 更新于 2020-08-16 分类于 骑行记

接着昨天的步伐，今天去了水之森公园。 其实今天才是事先定下来的计划，昨天从折立回来跟ヒメちゃん商量了一下 七点半从会馆出发。 其实早上出来的时候正在下雪，但还是努力地骑了过去。 额，这种小学生日记的展开是怎样⋯⋯ 总之到达公园的时间比想象早了很多。 其实公园的内容并没有什么好多说的。无非就是树林，水池，水池，树林。 偶尔有禁止进入的地方，但是也毫无感觉地过去了。 拍了挺多照片，但是基本上自己不太满意。 之后这样的时间不多了呢，毕竟我自己基本上只会选择城市的地方，森林对我来说，难度太大了。 其实不知道以后还能不能再见了。毕业以后。 那时候我会在哪儿呢。 我煞风景的能力真是一流啊。
