西面海洋

发表于 2010-04-28 更新于 2020-08-16 分类于 仲夏夜之梦

算算下来，又是一个夏天。

很不容易有这样舒服的天气，温度比较和适，阳光不刺眼，风不大，吹在身上暖洋洋的。

突然想到，很久没有用这种小学生般的词语了。

站在机械楼4楼的办公室，推开窗户看着窗外，风从小小的窗口吹进来，突然想到好像身在栾城。上一次有这种感觉忘了是什么时候了，大概是拿回家里的票那天（好像其实也没几天），坐上北安线，看着车上的人有些轻微的乡土气息，说着我听不懂的话，闭上眼睛，好像曹安路是那条土路，那辆车，号码是202.

果然这个才是学校的样子。身为同济从（这话说的），心中的大学只有两种样子，一是本部，那是人文气息；一是嘉定，这样的新校区，安静，清爽。说不清为什么用清爽这个词，但是，就是这样。

周围方圆数公里之内都鲜有高层，一眼望去就是明亮的阳光射来的方向雾蒙蒙的几个楼，一片辽阔，像一片海。夏天快来了，如果真的没有2012的话，几年后的夏天，就在这样的阳光下走也好。

接下来又不得不想起了明天的考试。如果不是有它。想想都开心，哇哈哈哈

但考试还是要有的，这就是杯具。

这个，是真不知道……

发表于 2010-04-21 更新于 2020-08-16 分类于 再不再四年

事实上，在我心情半好不好的今天，有两件事让我彻底闷下去了。

首先是锅炉作业。这个……错那么多我可以忍，毕竟作业是自己做的，错了就错了，只能怪自己不会，上课没好好听（虽然有很多是老师批改不认真才错的……有些题我真的觉得我没错）。但是我忍不了的是……额……老师那个态度。作业批改不认真也就算了，还在发下来之后一本正经地告诉我们这个占到25%的成绩。哇擦擦……早知道这样我们也去找答案去了。抱着这种奇怪的想法的老师甚至连第二次的作业也没有给我批。好吧好吧……这样我认了。

于是中午怀着满腔的郁闷以老师的名字打了一个小时的CS，外加啃了四个包子。然后这就是第二个杯具。

因为体育课跑1000米了。

本来以为大学生涯最后一次1000总会跑出点儿成绩来的，至少不是以这样一种样子落幕。呃……好吧我承认我一点儿心理准备也没有，加上中午的包子与空气作用搞得肚子不爽，两圈半酱油打下了，居然有4’12’’……完了伴有强烈的想吐出来的感觉。好吧时运不济命途多舛，今天我认栽。

倒是天气不冷，昨天刚下过雨场地都是湿的。跑完颓然地躺在人工草坪上热气从身子下面冒出来，想，呀，大学……哦不乃至人生中最后一次1000就这样结束了呢。果然不是一个典型的里程碑式的结局。

回来给某呆发短信说今天跑1000。然后说累了休息休息。不过想想去年这个时候她们跑完800的兴奋，我倒是真的挺闷的。或者说，我还真的不如跟有些人一样5月6号跑了。至少不会是这样一个非典型结尾。

回来坐了一会儿，突然开窍：哦天那……还有锅炉。MY GOD……

有道是，春雨贵

发表于 2010-04-21 更新于 2020-08-16 分类于 空の人生

一直在下雨，一直在下雨，就是什么都不想干。

刚才在翻地图册，看到本部附近，突然想到去年的春天，前年的春天。为什么感觉不一样了呢？

不过这样的感觉挺好的，除了冷一点儿。

有点儿想推开窗户大喊：“呀我什么都不想干什么都不想干呀~！”

然后听着那回声。

但是现实不允许，第一是这个地方不会产生回声，然后是如果我喊了，会被当成某种不正常的。

最最重要的是，该干的事情，还是要去干 。

另PS，我还是挺喜欢KYON这个角色的。不过看那个更像是看夏天一样。我不知道我的题目跟正文有什么关系，正如我不知道我的正文跟这个PS有什么关系一样。

哦乱……果然是春天吧
