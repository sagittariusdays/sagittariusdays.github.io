就那么一顿饭

发表于 2010-04-07 更新于 2020-08-16 分类于 再不再四年

现在一直觉得没有什么地方可去。就像这次的清明假期，基本……算是上海市区五日游，虽然最后还是累到走不动路。

最能算是去玩的就是周一那天，去了南翔。好吧，我承认是为了带某孩子去吃小笼的大于去玩的目的，南翔也没什么好玩的。

于是，带上周日买的钱包——我觉得算是一个不错的……嗯……礼物——以及包里面的确不比钱包价值的现金，乘上L11出发了。

虽然是第一次从曹杨路往下行方向坐，不过，感觉……人好多啊……

为了使我们的目的看起来尽量不像是只是去吃小笼，所以我们去了所谓的“古猗园”。也就是一个一般的小园林，感觉没什么太多，只是觉得……这个……翻修的也太新了吧……

于是中午去吃小笼。真的是挤不进去队伍，现在挺后悔的是在买小笼之前把相机收起来了。那队……从里面拐出来再拐啊拐，看起来很长，其实也挺快的，不过也花了近一个小时才拿到。就为了这一笼，吃完了，觉得也就那么一回事，好吃是好吃，不过还不值得去为此跑这么远然后再等上一个小时。不过无所谓啦~就是来玩的。

之后去下一目的地是步行过去的，后来证明这个决定很失策。因为走到差不多的位置的时候我们已经都走不动路了。不过，看那里人比等小笼的人还多……于是……算了吧

其实这就是一趟小笼之旅，吃的意义远大于去玩。不过，回来的时候……这回是真的谁都走不动啦

篮球？我们打的不是酱油

发表于 2010-03-30 更新于 2020-08-16 分类于 再不再四年

事实上是，我觉得有必要给今天的篮球赛说点儿什么。

曦哥语，这可能是四年里最后一次班级联赛。纪念总是有的。当时还没有分班的时候嘛，场上队员也不一定能到齐。不要说观众了。今天，基本能出动的都出动了，头一次把一个场地围满。要的是什么？就是个气势，造势其实大家都会的。

体育嘛，大家都重在参与，于是我们一群曾经习惯了在场上跑跑然后酱油打够就回家烧菜的人不那么热爱谈比分。毕竟比分不是一个队的事，更不是两个队的事，而篮球也不是五个人的篮球。

无兄弟，不篮球。一直很喜欢这句广告语，球场上五个人，加上教练替补训练员，这就是一家人，为同样的理想去打球。大家都想拿冠军的，但冠军只有一个。所以总归有要回家的。希望我们不是第一轮结束就可以，该去图书馆的去图书馆，该回寝室的回寝室。这就行了。

不管怎样，第一场结束了，还是套用小学初中高中等等的运动会用惯的句子，我们在一起，我们努力了，这就足够。

这一年，又一年

发表于 2010-02-05 更新于 2020-08-16 分类于 空の人生

那十天，那三天。
翻出来上一次写的东西，还是十一的时候。掐指算来，四个月，小半年。
中间的过程忽略不计，还是要从1月4号说起——恐怖的十天考试的开始，整整一个月前。
现在已经回忆起不多，但是总之还是那种白色恐怖笼罩着全身，每天不只一遍说慌，顺便手也拿不动东西，机械地看书本，机械地熬夜，机械地去考试。
于是，十天以后，解放了。结果归结果，至少，我付出的东西，远超过我所得到的。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
收拾好了东西，刚想要轻松地跟这个城市说再见，却得知了悲惨的消息。不晓得是什么原因吧，没有八卦的嗜好，愿逝者在天堂安息，那里没有作业，也没有考试。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
北上。在那个时候，我还不知道迎接的会是怎样的三天。现在想想，那么紧张，没必要。
她说的真对，下了火车换汽车，下了汽车再换车。如此的一个小“山村”。很漂亮。然后就总是有一种以后回家的感觉。
第一天还是挺紧张的。虽然最后还是很拘束，但是总还是有在家的感觉了。那个屋子，那样的阳光，想忘也忘不掉。
至于泰山与曲阜——玩得尽兴，也有遗憾，比如孔子故居比如十八盘。但是，在这样一种情况下出去旅游，无论是谁，都会很开心吧。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
再一次动车。没有想象中的动来动去。一惯地，坐在窗边数车站，从大雨数到阴天。当我放下杂志的时候，车已经跑在京津高铁上了。京津交界的地方，列车冲出那片乌云，顿时一片阳光灿烂，地上一条阴暗分明的影子，心情一片舒畅，还是那个城市，还是那个我。我，又回来了。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
回家。
车子从一个我不熟悉的方向进了京城，南站。
这是我第一次回到这个新建成的车站。巨大的弧顶，光滑的地面，圆形的候车室。是不是每个南站都是这样的设计。而我看到的，分明是几年前那群绿色的小平房。出站，一点。在出站口外柱子旁边看到那两个人，我知道，我到家了。
然后是一路聊天，当然少不了被指责为什么没有带人回来。其实这不是我的原因，我说了我倒是想呢，话说北京不是还有巧克力公园么。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
到家一片过年氛。自然要收拾屋子，一天到晚累得不成样子。但是，过年就是过年了，累一点儿是累一点儿，心里有个安慰。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
配了新眼镜，新年新气象，喜庆一点儿。还有张医生，去接你其实真的是顺便还有我怕散了瞳我真的回不来。去年的这个时候压力太大，哭了一场。没办法，按我妈的话，那是把这一年都哭背运了。但愿今年不会。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-眼睛又不行了。不能长时间看电脑。翻书，看去年冬天买的二战丛书，然后想到了很久很久以前，想到当初把盟军一玩通关的时候那种喜悦。那种心情，已经不会回来了。于是把盟军一二下了下来，于是认真研究所谓“战术”，于是好像又回到了以前。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-回到家却没看一场比赛。上午的时候一般都在打打扫扫。再加上对家里电视有很大的意见。好像回来以后没怎么打开过电视了。在曹的带领下变成了一个HOOPER，每天上去看看文章，翻翻图片，下下比赛。开着迅雷，在一边躺着，想想，又是一年过去了。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-进了二月，手机果然停机了。懒的去充钱了，过几天再说吧。想想今年2月14，大年初一。多好的日子。靖曾经说过，高中的每个情人节都是在学校，大学每个都是在假期，这很不利。那又怎么样呢。年要过，节日也要过。我说过寄巧克力过去给你，不会忘。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-一天一天数日子。盼着能哪天早点回去或者把某个孩子接过来。小李两口子进京了，HRT说要去找也一直没动，老白也不回来了。我还是每天对着太阳，干每天要干的活，准备过我的新年。
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
最后还是那些话。今天看到的一个视频让我更有了动力。你地铁规划到这儿吧，高铁12年也就开通了。我家的房子也快拆迁吧，要是顺手在划一下那就齐了。盼归盼，做梦归做梦。当然，做梦也要有个好梦。

Viva la vida

发表于 2009-10-07 更新于 2020-08-16 分类于 仲夏夜之梦

这一次过去，留下了这一首歌。其实我不知道是在什么时候提到的这首歌，大概是在散步的某个时候吧。
几天下来，身子像散了架一样（好像每次出去逛完之后就会有这样的感觉，而且总是会用这一句话来形容。）其实从客观上来说，对于一个以前没有来过某城市的人来说，两天半的时间很难完全了解一个城市，但足够对这里留下一个印象。我不知道给他们留下的是什么印象。对于我来说，像是梦中的两天。
其实最美中不足的就是天天的“三缺一”。不过这也不是我说了算的。而对于蜡像馆，不得不说是一个遗憾。其实说白了，剧情和以前都是一样，只不过换了个地方，而这个地方的特殊性才让这几天变得比以前有意思？意义。
…………
整整一天前，没错，一天前，才是真正的灵魂之所在。像是那么漫长的夜，却丢下一个人，熟睡在另一个人的怀里。很俗套的情节的相反，睡梦中那一首首歌静静地在耳边流过，是一个梦。现在，让我站着都能睡着。梦里，是那个梦。
…………
早上还是一片寂静。望着那个计价器出神。跳出车子看到的太阳，日出。很久没看了。飞机起飞的时候还在公交车上不省人事地大睡，阳光的早晨，城市中心，微风，长椅，这就是一切。出现在夏末的梦。
…………
踏上北安跨的时候心里空落落的。一条短信过去，回来的电话告诉我那条短信不是我发的。
应该不是吧，那是活在梦里的另一个我。
…………
似乎脚下的路由砖砌变成了柏油铺成，路的那一头在闪着绿灯。这就是方向，无论我能不能到达。
路是要走的，这是现实。
…………
耳边响起的却不是VIVA LA VIDA。但这首歌一定会在什么地方响起。现在。
不可能有红灯。因为我在前进。
