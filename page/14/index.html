考试的结尾，离别的钟声

发表于 2010-06-29 更新于 2020-08-16 分类于 再不再四年

十几天前看到过一个状态：四年后还有世界杯，四年后不一定有考试了。

这几天一直在世界杯的气氛下准备着考试，不过，没有像以前那个样子那么慌了。可能是因为供热开卷建筑设备给了范围。大四的都回家了，学校里一种说不出来的气氛。

谁知道今天供热开卷也是个杯具。不过现在已经没有什么感觉了。

考完试了就好，晚上还有荷兰比赛，在没有考试的日子里，这几天，还有世界杯。

我是真·伪球迷

发表于 2010-06-14 更新于 2020-08-16 分类于 再不再四年

总觉得轮回一事是确实有的，偶尔的一些事情回想到以前，反反复复的那个样子的梦想的夏天，然后一次一次地就那么平静地过去；数年一次的高潮低谷；若干个月份起伏不定的心情；还有，英格兰的门将。

按照某种分类方法我的确可以归并入“伪球迷”的庞大团体中。从不看俱乐部比赛只看四年一届的各种杯赛，对各个国家队的了解大于对各个俱乐部的了解再大于对中国队的了解。然后在这个基本上所有男孩男生男士男人老爷爷都当作节日的日子里，跟风，或者不是跟风地去看几场比赛，即使我连我喜欢的球队里球员的名字都说不全。

四年前的时候，学校没有电视，当时正赶上考试的时候，大家都在宿舍里，每天听听广播，跟能回家的同学聊聊各个队的情况，回家的时候赶场看了决赛，顺带还有荷兰对葡萄牙的回放——这场比赛给我留下了着实深刻的印象。当时对足球的了解不如现在（虽然现在也不怎么了解），只是在一个完全中立的立场看完了那些比赛。心里想，这比赛真XX的恶心。然后当两年后，我在大学里看完了欧洲杯小组赛荷兰所有三场比赛之后，我终于觉悟：那场比赛的裁判真XX的XX。

自08欧洲杯之后便开始喜欢上了荷兰队，像我看不懂篮球战术一样我也看不懂足球战术，但是就是觉得，呀这些人踢得好流畅，感觉好舒服的样子。特别是三战9-0，其中两场的对手是法国和意大利。然后我一直把那一年荷兰被俄罗斯淘汰的原因归结为——那一场我没看比赛。不过最后Esp捧杯这个结局还是让我比较欣慰的。虽然我只看了两场他们的比赛……额……对不住，时间安排的实在不好。

八年前也有那些比较疯狂的日子，伴随着——额——中国队首次冲进世界杯——随后三场丢掉9个球没有进球就这样被横扫了。记得当时还是很和谐的，那是举国欢庆的日子，作文课上老刘还在给我们拿出收音机来听直播。看完中国队出局，看完棒国让人大跌眼镜地做掉那么多强手然后冲进四强然后在三四名决赛中被土耳其闪电进球然后真正实力显现出来完败掉比赛，看完巴西捧杯，我去地摊上买了五块钱一张的D版盘和一个D版的飞火流星，之后一个假期沉浸在FIFA WC2002以及踢球砸坏别人家玻璃中。昨天晚上还和人聊起来这个，想到球飞起来还会带着火星，欧文的中场BUG，还是会想起那个天天飚车喝汽水的年代。怎么都已经那么多年了！

十二年前。好吧我承认我一直对运动一点兴趣都没有。什么是世界杯？我只知道球星卡。

现在回想起来在有些事情上我还是相当纠结的，就好比我最早接触的运动是足球却对它的兴趣一直一般，而高一才接触——真的只是“接触”而已——篮球，现在却一直在关注一样。可能是因为NBA30支队伍记起来比较容易，美国以外的联赛直接可以忽略，而足球那么多联赛那么多球队，我记不住，也懒得去记。虽然这样，一到杯赛的时候我总会乐呵呵地跑去看足球，NBA总决赛？黄绿大战？一边去吧。今天的主角是足球。

在知道学校将会转播每天的前两场的比赛之后还是有点欣慰的，至少不用像08年那样跑去饭店通宵看比赛了，再者说这荒郊野外的地方也没有饭店会给你通宵。结果这一点儿欣慰被“禁止带酒精饮料进入”这一条基本上算是毁了。虽然对我没什么影响，但是大家都在说“不带酒你看个什么球”，于是在这种言论下我只好跟风在宿舍里面挤在人堆里看球。虽然，可能过了今年这样看球的日子再也没有了。若干年之后，当2012不成真，世界杯再次到来的时候，我能叫上几个朋友来家里，重温一下这个年代看球的日子，说想当看哥支持XXXX结果XXXX，说XXX退役了现在队里都是新人，说XXX这个教练不行，这才是看球的乐趣。不过如果那个时候某人会跟我一起看球的话，就这样安安静静地看球也不错。虽然我认为，看足球还是要激情多一点才行。

这也许是大学四年间最热闹的节日了。怨念在于为什么这个会跟考试时间冲突，如果没有考试这就完美了。不过想想我们的考试，再想想学弟学妹们，便释然了。尽情地去欢呼吧，吵到复习这个事情已经跟我没有关系了。

总是在回忆之前，那种离愁别绪

发表于 2010-06-01 更新于 2020-08-16 分类于 仲夏夜之梦

好像所有的事情一遇上最后一次，总能给人一些伤感的感觉。就好像在梦里的十四天一样。

果然在上岗之前对这里是没什么感情的，然后果然在离开的时候，很多的话说不出来。熟悉岗位时对园区的陌生，与队友配合不默契，到最后一天的高峰岗，大家一班下来哑掉的嗓子，离开之前对园区和志愿者生活的眷恋，十四天，宛如梦幻。

5月16日第一次入园，30日最后一班岗，这半个月，经历过各种事情。各种帽子，各种参观者，各种问题，各种囧事。15天内的收集控，15天的半紧张半空闲。经历了第一次入园人数50万的高峰，边看数字猜测着今天会不会到40万，结果就这么被跃过去了。经历了30日的一些尴尬一些冲突；没有第一批他们站的时间长，没有第一批他们的福利好，但是我还是想说，我们第二批，是不输给第一批志愿者的。

进可攻，退可守。能源杀。这些车上的记忆，随着昨天最后一班车，成为了历史。最后一次，突然想到了很久没见的上海韩志强，最后一次看到18万吨，最后一次讲“基础搭讪学”还有那些名词，还有最后一次超越31号车。同学们，同伴们，谢谢你们。32号车的司机，谢谢你。

然后是最后一顿夜宵，味道果然比之前要好上一点点儿。脱掉那一身白菜，之后，我们就只会以普通的参观者的身份入园。还有最后一次没有换成徽章的遗憾，都在这最后一天，成为了历史。

昨天晚上便装在园内，见到一个人向我迎面走来。我本能地迎上前去，刚想开口，然后发现我身着便装。我已经不是志愿者了。随后就是一阵失落，就是这样，志愿者，结束了。

就这样不知道该说什么好了

发表于 2010-05-22 更新于 2020-08-16 分类于 再不再四年

我是一直不想跟风来写一些所谓的“世博日记”这一类的东西的。不过大家也都这么兴致勃勃地在一天一天的写。其实我觉得这种东西只要一个微博之类的东西就好了，一天一两句话，也不多，足矣。

话说回来，在里面的生活并没有多出彩，也并没有多无聊。但就此写出一天天的流水帐我还是没这个能力的。想当初高中作文的时候也是跟着来回来去胡乱煽情，别的也不会什么。不过也总是有一些好事儿的，比如，HRH the Prince of Orange，然后我们在他坐过的位置上来回坐了一遍。之后这两天无事，天天一班岗，然后享受天天被人称白菜的乐趣。

然后今天小活动了一下。晚上站岗的时候来了两个法国参观者，然后不会说英语，然后就悲剧了……我还是不知道有没有跟他们指示清楚，呀，失败。

还真是不知道该说什么好了，至少，这个世博会比我之前想象的要好，好很多。
